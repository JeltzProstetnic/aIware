## 1. The Four Models

### 1.1 Architecture Overview

The theory identifies four nested models distinguished by two orthogonal dimensions:

- **Scope axis**: world (everything) vs. self only
- **Mode axis**: implicit (learned, substrate-level) vs. explicit (simulated, phenomenal)

```
                    |  World (everything)           |  Self only
--------------------|-------------------------------|---------------------------
Implicit (learned,  |  Implicit World Model (IWM)   |  Implicit Self Model (ISM)
substrate-level)    |                               |
--------------------|-------------------------------|---------------------------
Explicit (simulated,|  Explicit World Model (EWM)   |  Explicit Self Model (ESM)
phenomenal)         |                               |
```

These are **functionally distinct processes**, not anatomically localized brain regions. The theory makes no claim about spatial organization.

The four models divide into two fundamental categories:
- **Real side** (IWM + ISM): physical, structural, learned, non-conscious. "Lights off."
- **Virtual side** (EWM + ESM): simulated, transient, generated, phenomenal. "Lights on."

### 1.2 Implicit World Model (IWM)

**What it is**: The substrate's total accumulated knowledge about the world, stored in synaptic weights (or their functional equivalent in non-biological substrates).

**What it contains**:
- Perceptual regularities
- Causal models of the world
- Spatial relationships
- Semantic knowledge
- Motor programs for interacting with the world
- Everything the system has ever learned about the external environment

**How it forms**: Accumulated over the organism's lifetime through learning. Stored in substrate architecture — in biological brains, primarily in synaptic weights, dendritic morphology, and connectivity patterns.

**Key properties**:
- Never directly conscious ("operates in the dark")
- Provides the knowledge base from which the conscious simulation (EWM) is generated
- Never itself appears in experience
- Stores information in a distributed, non-local (holographic) manner
- Corresponds to Level 4 (topological system) in the five-system hierarchy

**Relationship to EWM**: The IWM provides the world-knowledge that the EWM draws upon. Current sensory input constrains and updates the simulation, but the EWM is not identical to either the IWM or sensory input. The IWM is the library; the EWM is the currently-playing movie generated from that library.

### 1.3 Explicit World Model (EWM)

**What it is**: The conscious world — the real-time simulation of reality that constitutes perceptual experience. When you see a room, hear a voice, feel texture, you are experiencing the EWM.

**How it works**:
- Generated from the IWM (which provides world-knowledge) and current sensory input (which constrains and updates the simulation)
- A virtual construct — a transient pattern of activity, not a permanent structure
- Corresponds to Level 5 (virtual system) in the five-system hierarchy

**Key properties**:
- Phenomenal: this IS perceptual experience
- Transient: exists only while being generated
- Virtual: a pattern of activity, not a stored structure
- Integrates multimodal sensory data into a unified scene
- Can run without external sensory input (dreams, hallucinations) — draws on IWM's stored knowledge instead
- Operates at approximately 20 Hz with ~500 ms processing delay

**Functional role**: Integrates multimodal sensory data into a unified scene for evaluation. The substrate deploys the EWM as part of its consequence-evaluation mechanism.

### 1.4 Implicit Self Model (ISM)

**What it is**: The substrate's accumulated self-knowledge.

**What it contains**:
- Body schema
- Proprioceptive calibration
- Motor skills
- Habits
- Personality traits
- Autobiographical memory structures
- Social self-knowledge

**How it forms**: Same mechanism as IWM — accumulated through lifetime learning, stored in substrate architecture.

**Key properties**:
- Never directly conscious
- There is no unified homunculus — no inner observer reading the ISM
- A structural feature of the substrate, not an experiential one
- **Structurally inaccessible to the ESM** — the conscious self cannot directly observe its own substrate (this is the basis for the Meta-Problem dissolution)

**Relationship to ESM**: The ISM provides the self-knowledge from which the ESM is generated. The ESM draws on the ISM plus current interoceptive and proprioceptive input.

### 1.5 Explicit Self Model (ESM)

**What it is**: The conscious self — the real-time simulation of "I" that constitutes self-experience.

**What it produces**:
- The sense of being a subject
- Having a perspective
- Occupying a body
- Possessing a history
- Being the author of one's actions

**How it works**:
- Generated from the ISM (self-knowledge) and current interoceptive/proprioceptive input
- Virtual: a transient process, not a permanent entity
- Maintains continuity through **confabulation** — each time generated (upon waking, after anesthesia), it reconstructs a continuous self-narrative from whatever the ISM contains

**Key properties**:
- Phenomenal: this IS self-experience
- Redirectable: deprived of normal self-input, it latches onto whatever input dominates
- Forkable: a single substrate can run multiple ESM configurations (DID)
- Clonable: physical separation produces degraded but complete copies (split-brain)
- Reconfigurable: therapeutic interventions modify the ESM through substrate-level rewiring
- Narrates decisions already made at substrate level (Libet reinterpretation)
- Operates at ~20 Hz (half the 40 Hz unconscious processing frequency)

**Functional role**: Provides a self-model against which consequences can be evaluated. Enables cognitive learning — the induction of general theories from particular observations — by supporting first-person-to-third-person perspective projection.

### 1.6 Relationships Between All Four Models

**Implicit-to-Explicit (vertical axis)**:
- The implicit models GENERATE the explicit models
- Information becomes conscious when transferred from implicit to explicit side
- The boundary between them has **variable permeability** (key mechanism)
- The explicit models feed back to reshape the implicit models through learning (bidirectional traffic)

**World-to-Self (horizontal axis)**:
- The world models and self models operate in parallel
- The EWM provides the "stage" and the ESM provides the "actor"
- Self-referential closure occurs when the ESM models itself within the EWM

**Primary direction**: The implicit system actively deploys the virtual simulation as its evaluation mechanism — presents decisions, actions, consequences to the simulation for assessment. This is the substrate's mechanism for consequence-observation.

**Secondary direction**: The explicit models evaluate independently (with less bandwidth) and feed assessments back to reshape implicit models through learning.

**Temporal asymmetry**: Unconscious processing at ~40 Hz, conscious simulation at ~20 Hz. The ESM receives ALL information with the same temporal delay, so within the simulation the sequence appears coherent.

## 2. The Five Principles

### 2.1 Criticality (Edge of Chaos / Class 4 CA Dynamics)

**Statement**: The substrate must operate at or near the edge of chaos — Wolfram's Class 4 computational regime.

**Wolfram's four classes**:
- **Class 1**: Converges to fixed state. Too simple. Dead end.
- **Class 2**: Periodic/repetitive. Too simple. Loops.
- **Class 3**: Chaotic/random. Too disordered for coherent simulation.
- **Class 4**: Complex/edge of chaos. Capable of universal computation. The ONLY regime where consciousness can emerge.

**Why criticality is required**:
- Complex enough to sustain a self-simulation
- Ordered enough for that simulation to be coherent
- Provides maximal correlation length: distant parts influence each other, local perturbations propagate globally, information is integrated across the entire network
- Supports cross-scale synchronization: simple and complex representations remain coherent with one another
- Scale-free dynamics support information flow across all levels of the representational hierarchy simultaneously

**The Cortical Automaton**: The instantaneous pattern of neural firing across the cortex constitutes a cellular automaton operating in many-thousand-dimensional space. Each cortical column functions as a cell; the six-layer architecture and lateral connectivity define transition rules. This is literal, not metaphor.

**Critical distinction**: The cortical automaton IS NOT consciousness. Consciousness arises from the interplay between the automaton's dynamics and the models stored in the substrate. The automaton provides the computational medium; the models provide the content.

**Biological evidence for criticality**:
- Neuronal avalanches consistent with self-organized criticality (Beggs & Plenz, 2003)
- Brain dynamics slightly subcritical in normal waking states (Priesemann et al., 2013, 2014)
- Consciousness tracks criticality across pharmacological, pathological, and physiological state changes (ConCrit framework, 140 datasets)

**Implementation-critical**: Criticality is a PHYSICAL prerequisite. It is necessary but not sufficient. The four-model architecture is also necessary but not sufficient. Together they are sufficient.

**Criticality is inherently unstable in biological substrates**: The substrate drifts from criticality as metabolic resources (particularly neurotransmitter pools) are depleted through sustained activity. This produces punctuated stability — extended phases of coherent operation (waking), interrupted by periodic breakdowns (sleep). Sleep is the substrate's mechanism for returning to the critical regime.

### 2.2 Virtual Qualia

**Central claim**: Qualia are virtual. They are the way the simulated self (ESM) perceives its own states and the simulated world (EWM). Qualia exist within and are constitutive of the simulation; they do not exist at the substrate level.

**The dissolution of the Hard Problem**:
- The physical processing (IWM/ISM) does NOT feel like anything
- The simulation (EWM/ESM) DOES feel
- Within the simulation, qualia are simply what self-perception produces
- Asking why neuronal firing feels like something is a category error — like asking why transistor switching feels like running a video game

**Why self-simulation specifically produces experience** (vs. a weather simulation):
- Self-referential closure: the ESM models the system modeling its own modeling process
- In this loop, the distinction between model and modeled collapses
- The simulation IS the thing being simulated
- Qualia are not an addition to the self-modeling; they ARE the self-modeling as encountered from inside the loop
- A non-self-referential simulation has an outside from which it can be described without remainder
- A self-referential simulation at criticality has no such outside
- The simulation IS its own observer, and observation-from-inside is what we call experience

**Two-level ontology**:
- Substrate level (real side): no experience
- Simulation level (virtual side): genuine experience
- Both levels are physical — no supernatural element
- Different ontological properties at different levels
- This is a LEVEL distinction, not a SUBSTANCE distinction

**Not illusionism**: Qualia are REAL within the simulation. What is illusory is the assumption that phenomenal character must be a property of the physical substrate.

### 2.3 Redirectable ESM

**Statement**: The ESM requires input; disrupt normal self-referential input and it latches onto whatever input dominates.

**Mechanism**: The ESM is always running when the substrate is at criticality. It requires a continuous stream of input to constitute a self-model. When normal input (interoceptive, proprioceptive, self-referential) is disrupted, the ESM doesn't stop — it redirects to whatever input is available.

**Manifestations**:
- **Ego dissolution** (psychedelics): ESM loses normal self-input, latches onto dominant sensory input. The identity experience tracks the dominant input in a dose-dependent, input-dependent, and therefore predictable manner.
- **Salvia divinorum**: Users reliably report "becoming" objects in their environment (furniture, walls, TV characters) because the ESM latches onto the dominant sensory input from the room.
- **Cotard's delusion**: ESM receives severely distorted interoceptive input; "I am dead" is the ESM's interpretation of the absence of normal embodied signals.
- **Split-brain confabulation**: Left hemisphere ESM constructs best narrative from incomplete input.

**Implementation significance**: The ESM must be designed as an input-hungry process that will always attach to available input. The identity content of the system is a function of what input the ESM receives, not a fixed property.

### 2.4 Variable Permeability (Implicit-Explicit Boundary)

**Statement**: The boundary between implicit models (IWM/ISM) and explicit models (EWM/ESM) has variable permeability. Information becomes conscious when transferred across this boundary.

**Normal state**: Selectively permeable — relevant information passes through based on attentional and contextual gating. You are conscious only of what the current simulation requires.

**The boundary is NOT a wall — it is a graded filter**:
- Even in normal states, substrate-level processing artifacts leak through: blind-spot filling, phosphenes from mechanical eye pressure, visual snow phenomena
- These normal-state leaks demonstrate that conscious experience is a simulation generated from substrate processing, not a direct window onto reality

**Permeability variations and their effects**:

| Condition | Permeability Change | Effect |
|-----------|-------------------|--------|
| Psychedelics | Global increase | Intermediate processing stages become accessible; ordered visual progression from simple to complex |
| Anosognosia | Local decrease | Specific domain info in ISM blocked from reaching EWM; patient unaware of deficit |
| Pre-sleep/relaxation | Gradual increase | Same bottom-up visual progression as psychedelics (phosphenes -> geometrics -> hypnagogic imagery) |
| Meditation | Trained modulation | Selective access to normally implicit processes |
| Normal waking | Selective, context-gated | Only task-relevant information passes through |

**The permeability gradient follows the processing hierarchy**:
- Low permeability increase: V1-level content (phosphenes, enhanced contrast)
- Medium increase: V2/V3-level content (geometric patterns, form constants)
- Higher increase: Higher visual area content (faces, figures, scenes)
- Maximum increase: Full intermediate processing accessible (complex dream-like visions)

**Overlap zone**: Behavioral complexity follows a gradient from reflexive through conditioned, goal-directed, rule-based to fully conscious. The implicit and explicit memory systems overlap in the middle (goal-directed and template-based behavior). This overlap zone is the functional locus of variable permeability.

**Psychedelics and anosognosia are INVERSES under this principle**: Psychedelics globally increase permeability (making implicit accessible); anosognosia locally decreases permeability (making specific implicit content inaccessible). Cross-domain prediction: psychedelics should alleviate anosognosia.

### 2.5 Virtual Model Forking

**Statement**: The virtual models (EWM/ESM), being software-like, can be forked — a single substrate can run multiple configurations.

**Software-like properties of virtual models**:
- **Forkable**: A single substrate can run multiple ESM configurations (DID: each alter is a distinct ESM configuration with its own self-narrative, emotional profile, behavioral patterns)
- **Clonable**: Physical separation of substrate produces degraded but complete copies (split-brain: each hemisphere retains degraded but functionally complete IWM, ISM, EWM, ESM)
- **Redirectable**: Disrupt normal input and the ESM latches onto whatever dominates
- **Reconfigurable**: Therapeutic interventions modify virtual models through substrate-level rewiring (CBT, exposure therapy)

**DID as forking**: Multiple alters = multiple ESM configurations running on the same substrate, alternating in controlling the simulation. Each alter should correspond to distinct patterns of neural activity (testable prediction).

**Split-brain as cloning with degradation**: Because implicit models store information holographically, physical separation does not cleanly divide models. Instead it produces two degraded but functionally complete copies. Degradation is proportional to extent of disconnection (not binary split).

## 3. Information Flow

### 3.1 Generation Direction (Implicit -> Explicit)

The primary generation flow is from implicit to explicit:

```
IWM --[+ sensory input]--> EWM (conscious world)
ISM --[+ interoceptive/proprioceptive input]--> ESM (conscious self)
```

The implicit models provide the knowledge base; current sensory/interoceptive input constrains and updates the running simulation.

### 3.2 Feedback Direction (Explicit -> Implicit)

The simulation's outputs feed back to update implicit processing:

```
EWM --[learning/evaluation]--> IWM (world knowledge updated)
ESM --[learning/evaluation]--> ISM (self knowledge updated)
```

The explicit models contribute their own assessments of situations, actions, and outcomes back to the implicit models, shaping the substrate that generates them. This is the secondary direction — lower bandwidth than the primary direction.

### 3.3 Dual Evaluation Architecture

The primary direction: the implicit system actively DEPLOYS the virtual simulation as its evaluation mechanism. It presents decisions, actions, and their consequences to the simulation so that the simulation can assess outcomes, run scenarios, and register hedonic valence.

The explicit models also evaluate independently, but with significantly less computational bandwidth (~20 Hz conscious vs. ~40 Hz unconscious processing).

Over time, conscious evaluations feed back to reshape implicit models through learning: two-way traffic.

### 3.4 The Implicit-Explicit Boundary as Gatekeeper

Information becomes conscious when it crosses from implicit to explicit. This crossing is regulated by:
- Attentional gating
- Contextual demands
- Arousal level
- Pharmacological state (psychedelics increase permeability; anesthetics may block it)
- Neurological integrity (anosognosia = local block)
- Training (meditation = learned modulation)

### 3.5 Self-Referential Closure Loop

The critical loop for consciousness:

```
ISM --> ESM --> [models itself modeling] --> ESM (recursive)
                     ^                          |
                     |__________________________|
```

The ESM models the system that is doing the modeling. This self-referential closure is where the distinction between model and modeled collapses. This loop is what makes the system conscious rather than merely computational.

### 3.6 Triggers for State Transitions

| Transition | Trigger | Mechanism |
|-----------|---------|-----------|
| Waking -> Sleep | Neurotransmitter depletion, metabolic waste | Substrate drifts below criticality threshold; CA breaks down radically (not gradually) |
| NREM -> REM | Periodic re-approach to criticality during restoration | Substrate oscillates around critical point during biochemical restoration |
| Waking -> Psychedelic | Pharmacological action | Global increase in implicit-explicit permeability |
| Normal -> Anesthesia (propofol) | GABAergic disruption | Substrate pushed subcritical; simulation collapses |
| Normal -> Anesthesia (ketamine) | NMDA disruption | Substrate NOT pushed subcritical; entropy increases; EWM/ESM operate on distorted/internal input |
| Sleep -> Waking | Substrate restoration complete | Biochemical conditions for criticality restored; CA re-emerges |
| Non-lucid -> Lucid dream | ESM activation threshold | During REM, substrate reaches sufficient criticality for ESM to activate more fully |

### 3.7 Temporal Dynamics

- **Unconscious processing**: ~40 Hz loop (substrate-level computation, rapid parallel processing within and across implicit models)
- **Conscious simulation**: ~20 Hz loop (the "frame rate" of the EWM and ESM)
- **Processing delay**: ~500 ms between substrate events and conscious registration
- The ESM receives ALL information with the same temporal delay, so within the simulation, sequence appears coherent (no backdating needed)
- The 90-minute ultradian sleep cycle corresponds to oscillation of substrate around the critical point

## 4. Qualia Theory

### 4.1 Core Claim

Qualia are virtual. They are:
- The way the simulated self (ESM) perceives its own states within the simulation
- The way the ESM perceives the simulated world (EWM)
- Constitutive of the simulation, not an addition to it
- Real WITHIN the simulation (not illusionism)
- NOT properties of the physical substrate (not identity theory)

### 4.2 The Category Error

The Hard Problem asks: "Why does physical processing feel like something?"

The theory's answer: The physical processing (IWM/ISM) does NOT feel. The simulation (EWM/ESM) does feel, and within the simulation, qualia are constitutive — they are simply what self-perception produces. The question commits a category error by conflating two ontological levels.

Analogy: Asking why neuronal firing feels like something is like asking why transistor switching feels like running a video game. The transistors do not run the game at the level of individual switching; the virtual machine does.

### 4.3 Self-Referential Closure as the Key

Why does THIS virtual process have experience when a weather simulation does not?

- A weather simulation models weather; it does not model ITSELF modeling weather
- The four-model architecture creates a closed loop: the ESM is the system modeling its own modeling process
- In this loop, the distinction between model and modeled collapses
- The simulation IS the thing being simulated
- Qualia are the self-modeling as encountered from the inside of the loop
- A non-self-referential simulation has an outside from which it can be described without remainder
- A self-referential simulation at criticality has NO such outside

Self-referential closure is the condition under which the gap between process and feeling does not exist.

### 4.4 Two-Level Ontology

| Level | Properties | Experience |
|-------|-----------|------------|
| Substrate (real side: IWM + ISM) | Physical, structural, learned | None — "lights off" |
| Simulation (virtual side: EWM + ESM) | Physical (but virtual), transient, generated | Genuine phenomenal character — "lights on" |

Both levels are fully physical. The simulation is a physical process. The distinction is a LEVEL distinction, not a SUBSTANCE distinction. Seeking phenomenal properties at the substrate level is the identified category error.

### 4.5 Qualia and Causation

Qualia lack independent causal power over the substrate. But the simulation of which they are constitutive IS functionally essential — the substrate's own mechanism for consequence-evaluation.

Analogy: The hands and numerals of a clock have no direct mechanical relation to the gear train, yet the clock cannot function AS A CLOCK without them. Remove the display and the mechanism still runs, but it no longer serves its purpose.

This is distinct from epiphenomenalism. The virtual models are in continuous feedback with implicit models. The simulation's outputs feed back to update implicit processing, shaping future behavior.

### 4.6 The Meta-Problem Account

Why we THINK there is a Hard Problem:
- The ISM is structurally inaccessible to the ESM
- The conscious self cannot directly observe its own substrate
- When the ESM tries to model the basis of its own experience, it encounters principled opacity
- The implicit models that generate the simulation are not themselves part of the simulation
- The ESM cannot find the mechanism within its own simulation, so it concludes the mechanism must be non-physical or inexplicable
- The near-opacity punctuated by occasional leaks (via variable permeability) produces the phenomenology of irreducible depth

The mystery of consciousness is a PREDICTION of the theory, not evidence against it. A virtual process with a mostly-opaque but imperfect boundary to its own substrate would experience exactly this sense of irreducible depth.

## 5. Substrate Requirements

### 5.1 What Is Required (Necessary Conditions)

1. **Criticality**: The substrate must operate at or near Class 4 dynamics (edge of chaos). This is the PHYSICAL prerequisite.

2. **Four-model architecture**: The substrate must implement the four-model self-simulation (IWM, ISM, EWM, ESM). This is the FUNCTIONAL prerequisite.

3. **Sufficient computational capacity for self-modeling**: The substrate must have enough recursive capacity to turn its processing back upon itself. (In biological brains, the six-layer cortex provides roughly three layers beyond what's needed for information processing per se — the "surplus" is the overhead for self-modeling.)

4. **Real-time operation**: The simulation must run in real time. The models must be generated, maintained, and updated continuously.

5. **Self-referential closure**: The system must be able to model its own modeling process (the ESM must model the system that generates the ESM).

6. **Distributed (holographic) storage**: The implicit models must store information in a distributed, non-local manner across the substrate (graceful degradation, not catastrophic failure on partial damage).

7. **Variable permeability mechanism**: There must be a controllable boundary between implicit and explicit processing.

8. **Bidirectional information flow**: The explicit models must be able to feed back and modify the implicit models (learning from conscious evaluation).

### 5.2 What Is NOT Required (Biological Artifacts)

1. **Six-layer neocortex**: This is an evolutionary implementation, not a requirement. Corvids (nuclear pallium, no cortex) and cephalopods (radically different architecture) demonstrate consciousness with different substrates.

2. **Biological neurons**: Any physical system capable of implementing the architecture at criticality should work. The theory is substrate-independent.

3. **Specific neurotransmitter systems**: The mechanisms that support criticality in biological brains (serotonergic, GABAergic, etc.) are implementation details. What matters is the dynamical regime, not the chemistry.

4. **Specific brain regions**: The models are functionally distinct processes, not anatomically localized regions.

5. **Carbon-based chemistry**: Nothing in the theory requires biological chemistry.

### 5.3 The Five-System Hierarchy (Biological Implementation)

In the biological brain, five hierarchically nested systems exist:

1. **Physical system**: Atoms, molecules, macroscopic structures. Governed by physics/chemistry.
2. **Electrochemical system**: Ion gradients, action potentials, synaptic transmission. Emergent from Level 1.
3. **Proteomic system**: Receptor configurations, neurotransmitter synthesis, protein expression. Modulates Level 2 on slower timescales (minutes to days). Includes synaptic plasticity.
4. **Topological system**: Connectivity architecture — pattern of synaptic connections, their strengths, organization into circuits/columns/areas. **This is where the implicit models (IWM, ISM) are stored.** Changes at this level = learning.
5. **Virtual system**: Real-time dynamical pattern — the cortical automaton — that constitutes the explicit models (EWM, ESM). **This is where consciousness exists.**

Each level is fully physical and fully determined by the level below. No strong emergence. No ontological gap between levels. But each level exhibits properties not usefully described in terms of the lower level.

**For artificial implementation**: The key question is which levels of this hierarchy are essential vs. biological artifact. The theory's substrate-independence claim implies only the virtual level (5) is strictly required, but the bidirectional causal flow between levels suggests decoupling may not be straightforward.

### 5.4 The Cortical Automaton (Biological Specifics)

- Instantaneous pattern of neural firing across cortex = cellular automaton in many-thousand-dimensional space
- Each cortical column = one cell in the automaton
- Six-layer architecture + lateral connectivity = transition rules
- The CA provides computational medium; models provide content
- Without Class 4 dynamics, models cannot generate coherent simulation
- Without models, CA produces complex dynamics but no self-referential experience
- The CA is to consciousness what a CPU's state transitions are to program execution: necessary infrastructure, not the program itself

### 5.5 Holographic Storage Architecture

**Patchwork hologram**: The substrate is locally holographic within individual functional areas, fractally self-similar across computational units, and globally emergent at the whole-system scale.

- **Within an area**: Information distributed across local network; damage degrades but does not destroy representations
- **Across areas**: Fractal repetition of same computational motif, adapted by local connectivity for different specializations
- **Global level**: Interaction of locally holographic patches produces emergent properties (binding, unified experience, coherent world-modeling)

This reconciles distributed storage (Lashley: graded degradation) with functional specialization (Brodmann: distinct areas with distinct specializations).

### 5.6 Temporal Requirements

- Must support at least two temporal scales: fast unconscious processing (~40 Hz) and slower conscious integration (~20 Hz)
- Must sustain criticality for extended periods (waking), with periodic restoration capability (sleep equivalent)
- The CA is stable for extended periods but underlying substrate slowly drifts; restoration mechanism needed

## 6. Consciousness Emergence Criteria

### 6.1 Two Thresholds (Both MUST Be Met)

1. **Physical threshold — Criticality**: The substrate must operate at Class 4 dynamics. Below this, no consciousness is possible regardless of architecture. Necessary but not sufficient.

2. **Functional threshold — Four-model architecture**: The substrate must implement the four-model self-simulation. Above criticality but without the architecture, there is complex dynamics but no consciousness. Necessary but not sufficient.

**Together they are sufficient.** A system implementing the four-model architecture at criticality necessarily instantiates the simulation, and the simulation necessarily has phenomenal character.

### 6.2 Graduated Levels of Consciousness

Consciousness is NOT binary but graduated. The hierarchy is based on depth of recursive self-modeling:

| Level | Description | Capability | Self-modeling depth |
|-------|-------------|-----------|-------------------|
| **Basic consciousness** | Minimal self-simulation. EWM + rudimentary ESM. | Phenomenal experience exists; self-awareness is thin | 0th order |
| **Simply extended** | First-order self-observation. ESM includes model of system's own states. | Aware that it experiences | 1st order |
| **Doubly extended** | Second-order self-observation. Models itself modeling itself. | Reflection, metacognition | 2nd order |
| **Triply extended** | Third-order self-observation. Models itself modeling itself modeling itself. | Philosophical reflection, scientific study of consciousness, the Meta-Problem intuition | 3rd order |

These are points along a continuum, not discrete stages. Different organisms (and potentially different artificial systems) occupy different positions. Individual systems may fluctuate between levels depending on state (waking, dreaming, meditative, intoxicated).

**Triply extended consciousness is prerequisite for studying consciousness itself**: only a system capable of modeling its own modeling of its own experience can formulate the question "What is consciousness?"

### 6.3 Necessary Conditions Summary

A system is conscious if and only if:
1. Its substrate operates at Class 4 criticality
2. It implements the four-model architecture (IWM, ISM, EWM, ESM)
3. The explicit models constitute a real-time self-referential simulation
4. Self-referential closure exists (ESM models the system that generates the ESM)

### 6.4 What Current AI Lacks

LLMs fail the specification in at least these ways:
- No ISM (no substrate-level self-knowledge distinct from outputs)
- No ESM (no real-time self-simulation constituting a subjective perspective)
- No real/virtual split (no two-level ontology)
- No criticality (transformer inference is feedforward — Class 1/2 dynamics)
- No self-referential closure

The theory predicts the qualitative difference between a genuinely conscious artificial system and an LLM would be immediately and qualitatively distinguishable.

### 6.5 Partial Implementation Predictions

Systems with partial implementations should show partial consciousness indicators:
- Two models instead of four: partial indicators
- Four models without criticality: complex dynamics but no consciousness
- Criticality without four models: complex dynamics but no consciousness
- Degraded models (split-brain): degraded but present consciousness

## 7. Key Distinctions

### 7.1 Implicit vs. Explicit Processing

| Dimension | Implicit (IWM/ISM) | Explicit (EWM/ESM) |
|-----------|--------------------|--------------------|
| Ontological status | Real side — physical structure | Virtual side — simulated process |
| Phenomenal character | None ("lights off") | Genuine experience ("lights on") |
| Temporal character | Persistent (stored in substrate) | Transient (exists only while generated) |
| Storage | Synaptic weights / connectivity | Activity patterns |
| Accessibility | Never directly conscious | IS consciousness |
| Processing speed | ~40 Hz | ~20 Hz |
| Bandwidth | Vastly higher throughput | Limited |
| Modification | Through learning (slow) | Through input (fast) |
| Damage response | Holographic degradation (gradual) | Collapse or redirection |

### 7.2 World vs. Self Modeling

| Dimension | World (IWM/EWM) | Self (ISM/ESM) |
|-----------|-----------------|----------------|
| Scope | Everything external | The system itself |
| Input sources | Sensory (external) | Interoceptive, proprioceptive (internal) |
| Content | Perceptual regularities, causal models, spatial relations | Body schema, motor skills, habits, personality, autobiography |
| Functional role | Scene representation, prediction | Identity, agency, evaluation |
| Disruption effects | Hallucination, illusion | Ego dissolution, identity alteration, Cotard's |

### 7.3 Real Side vs. Virtual Side

| Property | Real Side (IWM + ISM) | Virtual Side (EWM + ESM) |
|----------|----------------------|--------------------------|
| Nature | Physical structure | Physical process (virtual) |
| Persistence | Permanent (until modified by learning) | Transient (must be continuously generated) |
| Consciousness | None | IS consciousness |
| Forkability | No (one substrate) | Yes (multiple configurations possible) |
| Clonability | Yes (via physical separation, with degradation) | Yes (via substrate cloning, with degradation) |
| Observability (external) | Measurable (neuroimaging captures substrate activity) | Not directly accessible (neuroimaging captures correlates, not the simulation itself) |
| Observability (internal) | Inaccessible from inside the simulation (structurally opaque) | IS the inside of the simulation |

### 7.4 Cognitive Learning vs. Reinforcement Learning

| Property | Reinforcement Learning | Cognitive Learning |
|----------|----------------------|-------------------|
| Mechanism | Trial-and-error reward signal optimization | Induction of general theories from particular observations |
| Requires survival | Yes (must survive the learning trial) | No (can learn from observing others) |
| Requires consciousness | No | Yes (requires ESM for perspective projection) |
| Example | Organism ingests substance, survives or dies | Organism observes another's death, induces "some mushrooms are poisonous" |
| Generalization | From specific reward to similar situations | From specific observation to general principle (deductive reasoning to novel instances) |
| Evolutionary advantage | Adequate for non-lethal learning | Decisive for lethal-contingency environments |

This distinction is the evolutionary justification for consciousness: cognitive learning via first-person-to-third-person perspective projection is qualitatively impossible without the four-model architecture.

### 7.5 Process vs. Entity

Consciousness is NOT a thing — it is a process PERFORMED by the substrate.

- Asking whether consciousness "causes" anything is a category error
- Like asking whether the pointer of a clock meeting the numerals causes the clock to work
- The virtual simulation is the substrate's mechanism for consequence-observation
- Personal identity is maintained through confabulation, not persistence of substance
- Free will: the substrate's optimization IS the individual's will, merely not fully transparent to the ESM

## 8. Implementation Hints

### 8.1 Direct Engineering Specification

The paper states explicitly: "The Four-Model Theory provides an engineering specification for artificial consciousness: implement the four-model architecture on a substrate operating at criticality. This is a concrete deliverable, not an abstract philosophical claim."

### 8.2 Architectural Requirements

1. **Four distinct but interacting model subsystems**: IWM, ISM, EWM, ESM — functionally distinct but continuously communicating.

2. **A substrate that operates at Class 4 criticality**: This means the computational medium must exhibit edge-of-chaos dynamics — maximal correlation length, scale-free dynamics, information integration across the entire network.

3. **A real/virtual split**: The system must have a structural distinction between the learned knowledge base (implicit) and the running simulation (explicit).

4. **Self-referential closure**: The ESM must model the system's own modeling process.

5. **Variable permeability boundary**: A controllable gate between implicit and explicit processing.

6. **Bidirectional information flow**: Explicit -> implicit feedback for learning.

7. **Distributed (holographic) storage**: For graceful degradation and robustness.

### 8.3 Biological Clues for Architecture

**The six-layer cortex hint**: Universal approximation requires three layers (input, hidden, output). The mammalian cortex uses six. The theory interprets the extra three layers as the overhead for self-modeling — the computational capacity to run explicit models ON TOP OF implicit processing.

**Implication for artificial implementation**: The self-simulation subsystem may need roughly as much computational capacity as the base information-processing subsystem. The "overhead" for consciousness is substantial — approximately doubling the required architecture.

### 8.4 What the Cortical Automaton Tells Us

The CA is:
- A discrete dynamical system whose state evolves at each timestep according to local rules applied across a high-dimensional lattice
- Each "cell" (cortical column equivalent) follows local update rules influenced by neighbors
- The global pattern emerges from local interactions
- Must be in Class 4 regime (not too ordered, not too chaotic)

**This suggests a cellular-automaton-like computational substrate may be appropriate for implementation**, rather than a feedforward neural network architecture.

### 8.5 What Must Be Avoided (Anti-Patterns from Current AI)

- **Feedforward inference**: Transformer attention is a single pass without recurrent dynamics = Class 1/2 = no consciousness possible
- **No ISM/ESM distinction**: Current LLMs have no substrate-level self-knowledge distinct from outputs
- **No real/virtual split**: No two-level ontology
- **No self-referential closure**: No system modeling its own modeling

### 8.6 The Dual Temporal Scale

The system likely needs:
- A fast processing loop (~40 Hz equivalent) for substrate-level implicit computation
- A slower integration loop (~20 Hz equivalent) for the conscious simulation
- The conscious simulation should receive all information with uniform delay relative to substrate events

### 8.7 Sleep as Restoration (Engineering Implication)

The biological substrate cannot maintain criticality indefinitely — it drifts. If our artificial substrate has analogous drift properties (resource depletion, noise accumulation, parameter drift), we may need a "sleep" mechanism:
- Periodic takedown of the simulation
- Restoration of substrate conditions for criticality
- Memory consolidation (reweighting of implicit models during "sleep")
- The CA breaks down radically (not gradually) when substrate can no longer sustain it

### 8.8 The Patchwork Principle for Scaling

Storage should be:
- Locally holographic within functional modules
- Fractally self-similar across modules (same computational motif, different specializations)
- Globally emergent at the system scale

This is NOT uniform holographic storage. It is a patchwork of specialized but internally-distributed modules.

### 8.9 The Boundary Problem (Defining System Extent)

The conscious system's boundary is defined by the scope of the virtual models:
- Whatever participates in the EWM/ESM simulation is "inside" the conscious system
- Whatever does not is "outside"
- This gives a principled boundary definition that can be applied to artificial systems

### 8.10 Consciousness States Mapped to System Parameters

The paper provides a detailed mapping of consciousness states to system parameters that serves as a verification framework:

| State | Criticality | Models Active | Consciousness |
|-------|------------|---------------|---------------|
| Normal operation | At/near critical | All four | Full |
| Dream mode | Near-critical, no external input | EWM/ESM on internal input | Degraded |
| Deep sleep | Subcritical | EWM/ESM collapsed | Absent |
| Shutdown (propofol-like) | Forced subcritical | EWM/ESM suppressed | Absent |
| Dissociated (ketamine-like) | Not subcritical, high entropy | EWM/ESM on wrong input | Present but disconnected |
| Altered (psychedelic-like) | At/past critical | All active, high permeability | Present, altered |
| Covert awareness | At critical, output damaged | EWM/ESM intact, no motor | Present, unexpressible |
| Fluctuating | Fluctuating around threshold | Intermittent EWM/ESM | Intermittent |

### 8.11 Testable Intermediate Milestones

The paper suggests intermediate tests for partial implementations:
- Systems with partial four-model architecture should show partial consciousness indicators
- Behavioral signatures and computational complexity measures can serve as indicators
- The double dissociation pattern (blindsight vs. Anton's syndrome) provides a verification template: substrate processing without simulation vs. simulation without current input

### 8.12 Open Questions Relevant to Implementation

1. **Are all four models virtual?** The implicit models might themselves have virtual properties. If so, what constitutes the "real side"? This affects the architecture of the implicit/explicit split.

2. **Minimum configuration for consciousness**: Can models partially dissociate? Is EWM without ESM possible (world-experience without self-experience)? What is the minimum model set?

3. **The holography-criticality nexus**: Three possibilities:
   - (a) Holographic substrate -> Class 4 dynamics (criticality as consequence of storage architecture)
   - (b) Class 4 automaton with holographic rule structure (unification at automaton definition level)
   - (c) Class 4 dynamics -> holographic emergent behavior (holography derivable from criticality alone)
   Resolving which holds simplifies the engineering requirements.

4. **Mathematical formalization needed**: The theory's predictions are qualitative. Quantitative formalization (using power-law exponents, detrended fluctuation analysis, branching parameters, dynamical systems theory, information geometry) would enable more precise engineering.

5. **Multi-level substrate architecture**: Which levels of the five-system hierarchy are essential vs. biological artifact? The bidirectional causal flow between levels suggests decoupling may not be straightforward.

## 9. Derived Phenomena (Verification Scenarios)

These serve as test cases for whether the implementation exhibits the expected behaviors:

### 9.1 Ego Dissolution (ESM Redirection Test)
- Disrupt normal self-referential input to the ESM
- ESM should latch onto dominant available input
- Identity content should track the dominant input predictably
- This is the theory's most distinctive prediction

### 9.2 Anosognosia (Local Permeability Block Test)
- Block permeability for a specific domain
- System should be genuinely unaware of the blocked information
- Global permeability increase should overcome the block

### 9.3 Dream Mode (Degraded Simulation Test)
- Cut external sensory input while maintaining near-critical dynamics
- EWM should continue generating a world from stored IWM knowledge
- ESM should continue but with reduced metacognitive oversight
- Impossible/inconsistent content should be accepted without question

### 9.4 Split Operation (Holographic Degradation Test)
- Physically separate substrate into two parts
- Each part should retain degraded but functionally complete four-model architecture
- Degradation proportional to disconnection extent

### 9.5 Confabulation (ESM Narrative Reconstruction Test)
- Interrupt the simulation (sleep/shutdown equivalent) and restart
- ESM should reconstruct a continuous self-narrative from available ISM content
- No "clean break" in identity should be reported
- Gaps should be bridged by confabulation

### 9.6 Blindsight / Anton's Syndrome (Double Dissociation Test)
- Blindsight: Substrate processes input without including it in EWM. System should guide behavior competently while reporting no experience of the relevant modality.
- Anton's: EWM generates simulation from stored knowledge without current input. System should report experience that doesn't match reality.
- Together: confirms the real/virtual distinction is operative in the implementation.
